Ruby exercises - part 4
Manipulation of FASTA and GTF files

Exercise

Problem 4a:
Add to the existing class AnnotatedTranscript the information about the nucleotide sequence of the transcript.

Now AnnotatedTranscript and AnnotatedGene should also provide the methods to_fasta and to_gtf for producing the corresponding two representations of the transcript/gene.

Question: is there a "smart" method for achieving such aim?
Hint: think about code re-use...

Prepare a simple program that builds an example for testing the implementation.

File: lab/examples/ex-4a-annotgene.rb

Exercise

Problem 4b:
Adapt the existing class GTFReader in order to read the sequence information from an additional FASTA file if is specified as an optional parameter.

Note 1: For this exercise, assume that all the transcripts are on the + strand.
Note 2: FASTA files are big, do not attempt to store all the genomic sequences in RAM.

Test your implementation using the GTF file ex-4-plus.gtf and the FASTA file of the genomic sequences ex-4.fa.
The resulting FASTA file should be identical to ex-4-transcripts-plus.fa.

File: lab/examples/ex-4b-gtfreader.rb
