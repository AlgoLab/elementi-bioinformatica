Visualizing genes and transcripts

Gene and transcript visualization

Full-length transcripts are often graphically represented as follows.


Where the blocks represent the exons while the lines/arrows represents the introns.

The "scale" at the top represents the genomic coordinates.

Programmatically draw transcripts from Ruby

There is a Ruby Gem that allows to produce similar transcripts diagrams in the SVG format.

Gem: bio-svgenes
Authors: Graham Etherington and Dan MacLean
Homepage: https://github.com/danmaclean/bioruby-svgenes
Docs: http://rubydoc.info/gems/bio-svgenes/frames/

bio-svgenes - Caveats

bio-svgenes v0.4.0 has some minor bugs.

While the authors fix them in the official distribution, we can apply some "monkey patches" to modify the code at runtime.

The patches are available in the file ex-7-bio-svgenes-patches.rb and can be applied at runtime by "requiring" the file as follows.

require 'bio-svgenes'
require_relative 'ex-7-bio-svgenes-patches'
